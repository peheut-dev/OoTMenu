# Menu Zelda : Ocarina of Time

Ce répo est celui lié à la vidéo [Comment ça marche ? Menu](https://youtu.be/CYscSugmOu0)

## Pré-requis

Unity 6 ou plus récent

## Installation

1 - Télécharger ou cloner ce répertoire sur votre PC
2 - Dans Unity Hub Ajouter le projet "Add > Add project from disk"
3 - Ouvrez le projet

## Plugin et library
- [DOTween](https://assetstore.unity.com/packages/tools/animation/dotween-hotween-v2-27676)
- Text Mesh Pro
